# 19-01: Create Utility Functions

Create helper function for generating initials from client names for avatar fallbacks.

Add getInitials helper function to packages/client/src/lib/utils.ts

**Implementation:**
```typescript
/**
 * Extract initials from a name string
 * @param name - Full name (e.g., "Emma Dubois", "Sound Production SARL")
 * @returns Initials (e.g., "ED", "SP")
 */
export function getInitials(name: string): string {
  if (!name || name.trim() === '') return '??';

  const parts = name
    .trim()
    .split(/\s+/)
    .filter(Boolean);

  if (parts.length === 1) {
    // Single word: take first 2 chars
    return parts[0].substring(0, 2).toUpperCase();
  }

  // Multiple words: take first char of first 2 words
  return parts
    .slice(0, 2)
    .map(part => part[0])
    .join('')
    .toUpperCase();
}
```

**Rationale:** Avatar fallback requires consistent initials generation across Grid/Kanban views.
