1. Dashboard (/)

5. Invoices (/invoices)
6. Quotes (/quotes)
7. Rooms (/rooms)
8. Equipment (/equipment)
9. Calendar (/calendar)

**Workflow pour CHAQUE page:**

1. Navigate to page:
   ```
   mcp__chrome-devtools__navigate_page({url: "https://recording-studio-manager.com/[page]"})
   ```

2. Wait for page load (2-3 seconds)

3. Take snapshot to list all interactive elements:
   ```
   mcp__chrome-devtools__take_snapshot()
   ```

4. Check console errors:
   ```
   mcp__chrome-devtools__list_console_messages()
   ```
   - Si errors trouvés → documenter dans ERRORS-FOUND.md

5. Check network requests for API errors:
   ```
   mcp__chrome-devtools__list_network_requests()
   ```
   - Chercher 401, 500, failed requests
   - Si trouvés → documenter dans ERRORS-FOUND.md

6. For each button/link in snapshot:
   - Click element:
     ```
     mcp__chrome-devtools__click({uid: "element-uid"})
     ```
   - Vérifier comportement (modal opens, navigation, etc.)
   - Si erreur → capturer screenshot:
     ```
     mcp__chrome-devtools__take_screenshot({filePath: ".planning/phases/3.4-comprehensive-site-testing/errors/error-XXX.png"})
     ```
   - Documenter dans ERRORS-FOUND.md

7. Take screenshot de la page complète:
   ```
   mcp__chrome-devtools__take_screenshot({filePath: ".planning/phases/3.4-comprehensive-site-testing/screenshots/[page-name].png"})
   ```

8. Update TEST-COVERAGE-MATRIX.md:
